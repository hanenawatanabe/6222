👑 题型一：PCA 灵魂三连推（出题率极高，必须默写）

*(考法出处：EE7403 24-25 Sem2 Q3)*

**【考试题目】**

给定一个包含 $n$ 个**已中心化**（均值为0）的训练样本的数据矩阵 $X$（大小为 $m \times n$）。已知协方差矩阵的特征值 $\lambda_i$ 和特征向量 $\phi_i$ 满足定义 $\Sigma \phi_i = \lambda_i \phi_i$。

**(a)** 证明：训练数据投影到单位特征向量 $\phi_i$ 上的方差 $v_i$ 等于特征值 $\lambda_i$。

**(b)** 证明：对应于两个不同特征值（$\lambda_k \neq \lambda_j$）的特征向量 $\phi_k$ 和 $\phi_j$ 是相互正交的（Orthogonal）。

**(c)** 推导：投影到这两个不同特征向量上的 1D 特征之间的协方差 $\sigma_{kj}^2$，并证明其结果为 0。

**【满分标准答案】**

**(a) 证明方差 = 特征值**



1. 投影后的数据为 $Y = \phi_i^T X$。
2. 投影方差 $v_i = \frac{1}{n}YY^T = \frac{1}{n}(\phi_i^T X)(\phi_i^T X)^T = \phi_i^T (\frac{1}{n}XX^T) \phi_i$。
3. 代入协方差定义 $\Sigma = \frac{1}{n}XX^T$，得到 $v_i = \phi_i^T \Sigma \phi_i$。
4. 因为 $\Sigma \phi_i = \lambda_i \phi_i$，所以 $v_i = \phi_i^T (\lambda_i \phi_i) = \lambda_i (\phi_i^T \phi_i) = \mathbf{\lambda_i}$ （单位向量模为1）。

**(b) 证明正交性 $\phi_k^T \phi_j = 0$**



1. 已知 $\Sigma \phi_k = \lambda_k \phi_k$ 和 $\Sigma \phi_j = \lambda_j \phi_j$。
2. 第二式左乘 $\phi_k^T$ 得：$\phi_k^T \Sigma \phi_j = \lambda_j \phi_k^T \phi_j$。
3. 第一式左乘 $\phi_j^T$ 得：$\phi_j^T \Sigma \phi_k = \lambda_k \phi_j^T \phi_k$。两边取转置（利用 $\Sigma = \Sigma^T$）得：$\phi_k^T \Sigma \phi_j = \lambda_k \phi_k^T \phi_j$。
4. 两式相减：$0 = (\lambda_k - \lambda_j) \phi_k^T \phi_j$。因为 $\lambda_k \neq \lambda_j$，所以 $\mathbf{\phi_k^T \phi_j = 0}$。

**(c) 推导特征协方差为 0**

协方差公式 $\sigma_{kj}^2 = \frac{1}{n} (\phi_k^T X)(\phi_j^T X)^T = \phi_k^T (\frac{1}{n}XX^T) \phi_j = \phi_k^T \Sigma \phi_j$。

代入 $\Sigma \phi_j = \lambda_j \phi_j$ 得到 $\lambda_j (\phi_k^T \phi_j)$。根据 (b) 结论 $\phi_k^T \phi_j = 0$，最终结果为 $\mathbf{0}$。

🧮 题型二：具体数值降维与误差最小化重构

*(考法出处：EE7403 22-23 Sem2 Q3)*

**【考试题目】**

一个数据集包含 4 个样本：$x_1=(1,6)^T, x_2=(4,7)^T, x_3=(2,9)^T, x_4=(5,10)^T$。

其协方差矩阵的特征向量已知为 $\phi_1=(1/\sqrt{2}, 1/\sqrt{2})^T$ 和 $\phi_2=(-1/\sqrt{2}, 1/\sqrt{2})^T$。

**(a)** 计算这 4 个样本在这两个特征向量上的 1D 投影特征 $a_i$ 和 $b_i$ ($i=1,2,3,4$)。

**(b)** 计算这两个特征值 $\lambda_1, \lambda_2$。

**(c)** 为了使**重构误差最小化**，请选择正确的单维度表示（$a_i$ 或 $b_i$），计算重构后的 2D 样本 $\hat{x}_i$。

**【满分标准答案】**

**【致命防坑点】：所有的 PCA 投影前，必须先减去均值！重构后，必须加回均值！**



1. **求均值**：$\mu_x = (1+4+2+5)/4 = 3$，$\mu_y = (6+7+9+10)/4 = 8$。均值 $\mu = (3,8)^T$。
2. **去均值 (Center data)**：$x_{c1}=(-2,-2)^T, x_{c2}=(1,-1)^T, x_{c3}=(-1,1)^T, x_{c4}=(2,2)^T$。

**(a) 1D 投影计算 ($y = \phi^T x_c$)：**



* **$a_i$ (投影到 $\phi_1$)**：$a_1 = -4/\sqrt{2}, \quad a_2 = 0, \quad a_3 = 0, \quad a_4 = 4/\sqrt{2}$
* **$b_i$ (投影到 $\phi_2$)**：$b_1 = 0, \quad b_2 = -2/\sqrt{2}, \quad b_3 = 2/\sqrt{2}, \quad b_4 = 0$

**(b) 计算特征值（利用方差公式 $\lambda_i = \frac{1}{n}\sum y_i^2$）：**



* **$\lambda_1$** = $\frac{1}{4} [(-4/\sqrt{2})^2 + 0^2 + 0^2 + (4/\sqrt{2})^2] = \frac{1}{4}[8 + 0 + 0 + 8] = \mathbf{4}$
* **$\lambda_2$** = $\frac{1}{4} [0^2 + (-2/\sqrt{2})^2 + (2/\sqrt{2})^2 + 0^2] = \frac{1}{4}[0 + 2 + 2 + 0] = \mathbf{1}$

**(c) 误差最小化重构：**

由于 $\lambda_1 > \lambda_2$，为了最小化误差，必须使用第一主成分 $a_i$ 和 $\phi_1$ 进行重构。

重构公式：$\hat{x}_i = a_i \phi_1 + \mu$



* $\hat{x}_1 = (-4/\sqrt{2}) \cdot (1/\sqrt{2}, 1/\sqrt{2})^T + (3,8)^T = (-2,-2)^T + (3,8)^T = \mathbf{(1,6)^T}$
* $\hat{x}_2 = 0 + (3,8)^T = \mathbf{(3,8)^T}$
* $\hat{x}_3 = 0 + (3,8)^T = \mathbf{(3,8)^T}$
* $\hat{x}_4 = (4/\sqrt{2}) \cdot (1/\sqrt{2}, 1/\sqrt{2})^T + (3,8)^T = (2,2)^T + (3,8)^T = \mathbf{(5,10)^T}$

🎨 题型三：高斯分布的几何作图

*(考法出处：EE7403 18-19 Sem2 Q4)*

**【考试题目】**

二维数据均值向量为 $[1, 2]^T$。协方差矩阵的特征值为 $\lambda_1=4, \lambda_2=1$，对应特征向量分别为 $\phi_1=[1/\sqrt{5}, -2/\sqrt{5}]^T, \phi_2=[2/\sqrt{5}, 1/\sqrt{5}]^T$。

要求在 $[x_1, x_2]^T$ 平面上画出一个椭圆来代表高斯 PDF，并明确标出椭圆的中心、长轴和短轴。

**【作图关键文字描述（考试时照着画）】**



1. **中心 (Center)**：将椭圆中心标在坐标系上的 **$(1, 2)$** 处。
2. **长轴 (Major Axis)**：由于 $\lambda_1 > \lambda_2$，长轴沿着 $\phi_1$ 的方向。穿过点 $(1,2)$ 画一条斜率为 $-2$ 的直线段。长轴长度比例为 $\sqrt{\lambda_1} = 2$。
3. **短轴 (Minor Axis)**：短轴沿着 $\phi_2$ 的方向。穿过点 $(1,2)$ 画一条斜率为 $0.5$ 的直线段，垂直于长轴。短轴长度比例为 $\sqrt{\lambda_2} = 1$（即长短轴长度比为 2:1）。

🧩 题型四：高维数据集的特征向量化

*(考法出处：EE7403 21-22 Sem2 Q4)*

**【考试题目】**

一个数据库有 $c$ 个类，每类 $n$ 个样本，每个样本 $m$ 个特征。元素表示为 $x_{kij}$。

**(a)** 给出如何利用 $X_{kj}$（第 $k$ 类第 $j$ 个样本的列向量）构造一个**均值为零**的类条件数据矩阵 $X_k$。

**(b)** 用构造好的 $X_k$ 写出协方差矩阵 $\Sigma_k$ 的表达式。

**【满分标准答案】**

**(a) 构造零均值数据矩阵：**

首先求出第 $k$ 类的均值向量：$\mu_k = \frac{1}{n} \sum_{j=1}^n X_{kj}$。

然后，将该类中所有的 $n$ 个样本分别减去均值，并排拼成一个 $m \times n$ 的大矩阵 $X_k$：

$$X_k = [X_{k1}-\mu_k, \ X_{k2}-\mu_k, \ \dots, \ X_{kn}-\mu_k]$$

**(b) 写出协方差矩阵：**

直接套用矩阵乘法定义，极大简化运算表达式：

$$\Sigma_k = \frac{1}{n} X_k X_k^T$$
